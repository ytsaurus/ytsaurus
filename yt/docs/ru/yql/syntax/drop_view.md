# DROP VIEW

`DROP VIEW` удаляет независимое [представление](select/view.md).

## Синтаксис

```yql
DROP VIEW [IF EXISTS] [cluster.]`//path/to/view`;
```

Если `//path/to/view` не является представлением, команда завершается с ошибкой.
Попытка удалить несуществующее представление также завершается с ошибкой, если не указан модификатор `IF EXISTS`.

## Доступность

`DROP VIEW` доступен начиная с версии языка [2025.05](../changelog/2025.05.md).

## Примеры

```yql
DROP VIEW `//path/to/view1`; -- удаляет представление на текущем кластере
DROP VIEW IF EXISTS cluster.`//path/to/view2`; -- явное задание кластера, завершается без ошибки, если `//path/to/view2` не существует
```
