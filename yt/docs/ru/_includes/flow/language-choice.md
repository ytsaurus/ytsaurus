## Выбор языка {#choose-language}

Flow поддерживает четыре языка для реализации бизнес-логики:

- **[C++](../../flow/cpp/getting-started.md)** — нативная реализация, максимальная производительность, полный контроль. Рекомендуется для высоконагруженных пайплайнов.
- **[Java и Kotlin](../../flow/java/getting-started.md)** — работают через механизм [компаньонов](../../flow/concepts/companion.md). Поддерживают Spring Boot. Подходят для команд с JVM-стеком.
- **[Python](../../flow/python/getting-started.md)** — работает через механизм [компаньонов](../../flow/concepts/companion.md). Самый простой способ создать прототип пайплайна или обрабатывать небольшой поток данных.
- **[YQL](../../flow/yql/getting-started.md)** — декларативное описание пайплайна в виде SQL-запроса. Минимальный порог входа, не требует написания кода на C++, Java, Kotlin или Python. Находится в активной разработке, ещё не вся запланированная функциональность доступна.
