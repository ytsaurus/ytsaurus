# Управление правами доступа

Основная информация о правах доступа и их типах описана в разделе [Права доступа](../../../../../user-guide/data-processing/chyt/cliques/access.md).

## Как выдать права на ресурсы клики { #instruction }

1. Откройте интерфейс клики, как описано в разделе [Как перейти в интерфейс клики](../../../../../user-guide/data-processing/chyt/cliques/ui.md#where).
1. На [Панели вкладок](../../../../../user-guide/data-processing/chyt/cliques/ui.md#tabs) выберите вкладку **ACL**.
1. Нажмите кнопку **Request Permissions** (доступна ответственному или администратору клики).
1. В открывшейся форме в разделе *Permissions* выберите необходимые опции: `use`, `manage`, `read` и `remove`.
1. В разделе *Subjects* укажите пользователей или группу, которым нужно предоставить права.
