# Получение версий CHYT и {{clickhouse}}

Версии компонентов {{product-name}} — это важные параметры для стабильной и согласованной работы всей системы. Несовместимость версий может вызвать ошибки при выполнении операций. Поэтому определение актуальных версий компонентов помогает быстро выявить источник проблемы.

Определить версии можно через:

{% list tabs %}

- Веб-интерфейс

    1. Перейдите в [веб‑интерфейс](../../../../../user-guide/data-processing/chyt/cliques/ui.md#where) клики.
    1. В блоке характеристик найдите параметр **YT operation Id** и перейдите по указанной в нём ссылке.
    1. В открывшемся интерфейсе Vanilla‑операции слева, на *Панели вкладок* выберите вкладку **Details**.
    1. В разделе **Description** найдите блок **Artifacts**.
    1. Чтобы увидеть полный список параметров, нажмите кнопку **Show more**.
    1. В списке параметров найдите поле `ytserver-clickhouse`. Внутри него указаны:
        - `ch_version` — версия {{clickhouse}};
        - `version` — версия клики CHYT.

- Запрос в клику

    1. Перейдите в веб-интерфейс Query Tracker: нажмите ![sql](../../../../../../images/sql-btn.png){width=24 height=24} в правом верхнем углу, в блоке [Кнопки действий](../../../../../user-guide/data-processing/chyt/cliques/ui.md#action-menu).
    1. Убедитесь, что вверху на панели выбран движок CHYT.
    1. В открывшемся окне введите и выполните SQL-запрос:

    ```sql
    SELECT chytVersion(), version();
    ```

{% endlist %}

{% note tip "Совет" %}

Если вы обращаетесь в техническую поддержку по поводу проблем с кликой, заранее уточните и сообщите специалисту поддержки версии компонентов. Это ускорит диагностику и поможет быстрее найти решение.

{% endnote %}
