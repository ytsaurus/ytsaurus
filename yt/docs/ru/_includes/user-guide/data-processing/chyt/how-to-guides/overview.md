Если вы только начинаете работу — рекомендуем начать с раздела [Жизненный цикл клики](../../../../../user-guide/data-processing/chyt/cliques/overview.md#lifecycle). Это поможет понять, какие шаги нужно пройти, чтобы подготовить клику к выполнению SQL-запросов, и что происходит на каждом этапе.
