## Changes to the built-in functions

Added the `LinearDestroy` function for consuming instances of linear types when their value isn't needed.

## Changes to the DateTime module {#datetime-module}

Added a new flag for printing the timezone offset with a colon to the `Datetime::Format` function.
Added a new specifier for reading the timezone offset to the `Datetime::Parse` and `Datetime::Parse64` functions.

## Changes to the Yson module {#yson-module}

Added the `Yson::Iterate` function for traversing subtrees of Yson nodes.
Added the `Yson::As*` and `Yson::TryAs*` functions for converting a Yson node to a specified type.
Added functions for [in-place editing](../udf/list/yson.md#yson-modify) of Yson nodes.
