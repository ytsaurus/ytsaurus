# Access permissions management

Basic information about access permissions and their types is described in the [Access permissions](../../../../../user-guide/data-processing/chyt/cliques/access.md) section.

## How to grant permissions to clique resources { #instruction }

1. Open the clique interface as described in the [How to open the clique interface](../../../../../user-guide/data-processing/chyt/cliques/ui.md#where) section.
1. On the [Tabs panel](../../../../../user-guide/data-processing/chyt/cliques/ui.md#tabs), select the **ACL** tab.
1. Click **Request Permissions** (available to the clique owner or administrator).
1. In the form that opens, in the *Permissions* section, select the required options: `use`, `manage`, `read` and `remove`.
1. In the *Subjects* section, specify the users or group to whom you want to grant the permissions.
