# YTsaurus Flow

This folder contains the source code of an early preview version of YTsaurus Flow, a framework for real-time data processing in the YTsaurus ecosystem.

It is not officially released and is not covered by any commercial support or similar arrangements.

If you are a Yandex employee, refer to the [internal documentation](https://nda.ya.ru/t/f1awNqa27Za5hk).
