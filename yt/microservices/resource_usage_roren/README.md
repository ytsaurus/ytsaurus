### Аргументы командной строки:
* имя кластера, свободный аргумент. Если не задано, то берётся из переменной окружения YT_PROXY.
* `--source` $-$ директория с входной(ыми) таблицей(ами)
* `--destination` $-$ директория, куда будет записана одна выходная таблица
* `--node-id-dict` $-$ путь до таблицы node_id
* `--node-id-dict-cluster` $-$ названия кластера, на котором лежит таблица node_id
* `--snapshot-id` $-$ может быть одним из:
    * `all`. Тогда берутся все таблицы из директории `source`, у которых есть суффикс `_unified_export`. 
    * `latest`. Тогда берётся таблица по симлинку latest, симлинк должен лежать в директории source. У названия таблицы должен быть суффикс `_unified_export`.
    * название (назовём его `name`). Тогда берётся таблица с названием `{name}_unified_export`.
* `--force` $-$ аргумент без значения. Если он не задан, то из множества таблиц с прошлого пункта вычтутся те, для которых в `--destination` уже есть результаты.
* `--pool` $-$ устанавливает pool во всех спеках.
* `--network-project` $-$ устанавливает network_project в спеке map-операции.
